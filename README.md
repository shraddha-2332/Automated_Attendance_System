# Smart Classroom Attendance System

A complete face recognition-based attendance system with external USB camera support.

## Features

- *Real-time Face Recognition* using external USB camera
- *Automatic Attendance Tracking* with session management
- *Web Dashboard* for teachers to manage sessions
- *Arduino Integration* for entry/exit detection (optional)
- *SQLite Database* for data storage
- *Manual Override* capabilities
